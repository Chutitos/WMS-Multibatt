# Reglas WMS

- Cada operación crítica debe dejar rastro: usuario, fecha, entidad, estado anterior y estado nuevo cuando aplique.
- Stock disponible, reservado, preparado y entregado no deben confundirse.
- Validar FIFO si existen lotes o fechas de recepción.
- Las vistas de bodega deben favorecer rapidez y baja tasa de error.
