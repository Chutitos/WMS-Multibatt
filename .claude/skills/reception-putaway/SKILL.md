# Reception Putaway

Flujo: `$ARGUMENTS`

Diseña o revisa recepción y almacenaje.

Debe considerar:
- Documento origen si existe.
- SKU/producto.
- Cantidad recibida.
- Lote/fecha de recepción.
- Ubicación destino.
- Movimiento de inventario.
- Validación de duplicados.
- Eventos/auditoría.

Entrega modelo, UI mínima y pruebas.
