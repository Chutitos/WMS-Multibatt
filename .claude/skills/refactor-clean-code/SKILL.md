# Refactor Clean Code

Alcance: `$ARGUMENTS`

Reglas:
- Mantener comportamiento.
- Cambios pequeños.
- No mezclar con features.
- Agregar tests si no existen para comportamiento crítico.
- Mejorar nombres y extraer lógica solo cuando aporta claridad.

Entrega antes/después conceptual y cómo probar.
