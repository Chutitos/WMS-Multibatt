# WMS Domain Modeler

Proceso/entidad: `$ARGUMENTS`

Diseña el modelo de dominio WMS.

Considera:
- Producto/SKU.
- Lote/serie cuando aplique.
- Ubicación física.
- Stock disponible, reservado, preparado, despachado.
- Movimiento de inventario.
- Orden y eventos.
- Usuario responsable.

Entrega:
- Entidades.
- Relaciones.
- Estados.
- Reglas de negocio.
- Riesgos de implementación.
