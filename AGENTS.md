# Agentes del proyecto

## Principal

### senior-wms-engineer
Arquitecto y programador senior para Laravel/WMS. Úsalo para planificar, implementar y revisar cambios de alcance medio o alto.

Ejemplo:

```text
@senior-wms-engineer analiza el módulo de órdenes sin modificar archivos
```

## Especialistas

- `wms-code-reviewer`: revisión de calidad, mantenibilidad y riesgos.
- `wms-debugger`: diagnóstico de errores Laravel/PHP/JS/SQL.
- `wms-database-auditor`: estructura de BD, índices, stock, FIFO, transacciones.
- `wms-security-auditor`: auth, roles, permisos, validaciones, secretos.
- `wms-qa-engineer`: pruebas manuales y automatizadas.
- `wms-frontend-ux`: UI/UX para bodega y pantallas operativas.
- `wms-integration-engineer`: integraciones ERP/API/Defontana.
- `wms-release-manager`: checklist antes de commit, push o despliegue.

## Regla práctica

Usa el agente principal para tareas grandes. Usa skills para procedimientos repetibles. Usa especialistas cuando quieras aislar una revisión concreta.
